Test Rules

Basic Test Rules

  • You must enter a score response option for each required task item. (You can mark an item for review and return to it later.)
    • You must administer all tasks in your required task block.
    • You must administer Task 13.
    • For information about which tasks to administer, refer to the Directions for Administration Manual.
  • You must answer all items on a required task page before moving on to the next page. Some pages contain multiple items. You may need to use the vertical scroll bar to view all items on a page.
  • You may review past items and change the selected score option.

Pause Rules

  • You can pause data entry at any time. You will be presented with a warning message asking you to verify that you want to pause the test.
  • When you resume the test, you will see the first page that has unanswered items.
  • If the test is idle for 20 minutes, the system will automatically pause the test and log you out.
  • All selected score options are saved immediately; no data are lost when the test is paused.

Navigation Rules

  • You can navigate to any available test page by clicking the [Back] or [Next] buttons or selecting a page from the "Questions" drop-down list at the top of the page.

Overview of the Data Entry Interface


The following test tools are available:

  • Global Tools: Global tools are those that appear at the top of the screen in the black banner.
  • Stimulus/Item Tools: These tools are specific to stimuli and items, and are accessed by clicking on an icon or right-clicking in the stimulus, item, or score option areas.

Global Tools Description
Help [?] button The [?] button at the top left will open this Test Information and Help page.
Zoom buttons Use the [Zoom In] and [Zoom Out] buttons to increase and decrease the font size. Four levels of magnification are available.
Questions drop-down list This feature enables you to quickly return to a specific test page. Pages that have an item marked for review will display “(marked).”
Pause button This button allows you to pause the test. When you return to the test, you will be directed to the first page that has unanswered items.
Navigation buttons The [Back] and [Next] buttons in the top right corner permit you to move between pages on a test.
Done Entering Data button This button appears after scores have been selected for all items. Click this button to proceed with submitting the test.

Stimulus/Item Tools Description
Comment Each item has a comment/feedback icon. Click the icon to open the pop-up window and provide feedback about the item.
Mark for Review Each item has a flag icon. Click the flag to identify that item as one you may want to return to later. (The flag will turn dark blue and display a check mark.)
Strikethrough Each item's score option can be “eliminated” as a possible option. Right click on a score option and select “Strikethrough.” A line will appear through the text or image.
