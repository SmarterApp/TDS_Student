Test Rules

Basic Test Rules

  • You must enter a response option for each required task item.
    • For the grades 4 and 6 social studies tests, you must answer all items for a task before moving on to the next page. Tests can be submitted once scores have been entered for all items.
    • For the English language arts, mathematics, science and OGT social studies tests, you must enter data for one minimally required task block (1-5, 3-9, or 6-12) and Task 13. For information about minimally required task blocks, please see the Data Entry Interface User Guide.
  • You may review past items and change the selected score option.

Pause Rules

  • You can pause data entry at any time. You will be presented with a warning message asking you to verify that you want to pause the test.
  • When you resume the test, you will see the first page that has unanswered items.
  • If the test is idle for 20 minutes, the system will automatically pause the test and log you out.
  • All selected score options are saved immediately; no data are lost when the test is paused.

Navigation Rules

  • You can navigate to any available test page by clicking the [Back] or [Next] buttons or selecting a page from the "Questions" drop-down list at the top of the page.

Overview of the Data Entry Interface


The following test tools are available:

  • Global Tools: Global tools are those that appear at the top of the screen in the black banner.
  • Context Menu Tools: Menu tools are for passages and items. To use these tools, open the context menu [].

Global Tools Description
Help [?] button The [?] button at the top right will open this Test Information and Help page.
Zoom buttons Use the [Zoom In] and [Zoom Out] buttons to increase and decrease the font size. Four levels of magnification are available.
Questions drop-down list This feature enables you to quickly return to a specific test page. Pages that have an item marked for review will display “(marked).”
Pause button This button allows you to pause the test. When you return to the test, you will be directed to the first page that has unanswered items.
Navigation buttons The [Back] and [Next] buttons in the top left corner permit you to move between pages on a test.
Done Entering Data button This button appears after scores have been selected for all items. Click this button to proceed with submitting the test.

Stimulus/Item Tools Description
Comment Each question has a tool for providing feedback about the question. Open the context menu and select [Comment].
Item Reset Open the context menu and select [Item Response Reset] to clear the entered response for an item.
Mark for Review Open the context menu and select [Mark for Review]. The question number will change and show a white flap in the upper right corner.
Strikethrough Each item's score option can be “eliminated” as a possible option.
  • Option 1: Select an answer option and then open the context menu and select [Strikethrough]. A line will appear through the text or image.
  • Option 2: Right-click on an answer option and select [Strikethrough]. A line will appear through the text or image.
