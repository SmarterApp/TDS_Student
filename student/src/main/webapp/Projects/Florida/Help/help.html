Overview of the Student Testing Site and Test Tools


The following test tools are available:

  • Global Tools: Global tools are those that appear at the top of the screen in the dark gray banner.
  • Context Menu Tools: Menu tools are for passages and questions. To use these tools, click the icons above the question, or right-click in the area of the passage or question.

Global Tools Description
Help [?] button The [?] button at the top right will open this Test Instructions and Help Guide screen.
Zoom buttons Use the [Zoom In] and [Zoom Out] buttons to increase and decrease the size of text and graphics within a single test page. Four levels of magnification are available.
Line Reader Use the [Line Reader] button to highlight a single line of text at a time in passages and items.
Notes button Click the [Notes] button to open an on-screen notepad. You can enter notes for yourself and save them. (Notes appear only for the test opportunity you are taking.)
Note: This tool is available for ELA Reading and ELA Writing tests.
Calculator button Click the [Calculator] button to open the on-screen calculator.
Note: The calculator is available for Grades 7–8 Mathematics and End-of-Course (EOC) Mathematics tests.
Questions drop-down list This feature enables you to quickly return to a specific test page. Questions that were marked for review will display “(marked).”
Save button This button allows you to manually save technology-enhanced questions. This is an optional feature. If you do not click this button, your answers will still be saved automatically.
Pause button This button allows you to pause and exit the test.
Navigation buttons The [Back] and [Next] buttons in the top left corner permit you to move between pages on a test.
  • For Mathematics tests and ELA Reading tests, clicking the navigation buttons will permit you to move between questions.
  • For ELA Writing tests, clicking [Next] will prompt you to end the test.

Note: When multiple questions are grouped together with the same stimulus, a button for each question number appears below the zoom buttons. You can click these buttons to navigate between the individual questions in the group.
End Test button Click this button to begin the process of submitting the test.

Context Menu Tools Description
Expand/Collapse
Passage		 Some test pages have a passage or activity on the left side of the page. You can expand this section.
  • Click the double arrow button [] to expand the left section so that it takes up most of the screen.
  • Click the double arrow button again to collapse the passage or activity.
Notepad Each Mathematics item has a Notepad tool. Select the icon that looks like a speech bubble. The Notepad window will open.

This tool is available for Mathematics tests.
Highlighter Highlight a section of text in a passage or test question.
Select text on the screen, click the context menu, and select [Highlight Selection]. The selected text will become yellow. To remove highlighting from an item, right-click on the item and select [Reset Highlighting].

Notes:
  • Text in images cannot be highlighted.
  • Highlighting is only available during a single test session. When you return to the test, any highlighting you did in a previous session will no longer be visible.
Mark for Review Select the icon that looks like a flag. The flag will become dark blue and display a check mark. To unmark a question, click the flag icon again.
Strikethrough For multiple-choice questions, you can "eliminate” an option and focus on what you think may be the right options.
  • Option 1: Select an answer option and then click the context menu and select [Strikethrough]. A line will appear through the text or image.
  • Option 2: Right click on an answer option and select [Strikethrough]. A line will appear through the text or image.
Tutorial Each question has a tutorial video that you can watch to learn more about that type of question. Select the round [ i ] button. The video will open on the screen.

Basic Test Rules

  • You can answer a question and mark it for review, and then return to marked questions later to change or review your answer.
  • Some questions are grouped together. You may need to use the grouped item navigation buttons in the upper right corner to view all questions in a group.

Accessing Context Menu Tools

The following tools are available in the context menu. You can access this menu by right-clicking on a test element, such as a stimulus, item stem, or answer option.

  • Highlighter
  • Mark for Review
  • Notepad
  • Strikethrough
  • Tutorial

Two-Button Mouse or Track Pad

If you are using a computer with a standard two-button mouse or a laptop with a two-button track pad, you can use the “right-click” button.

  1. Right-click anywhere on a stimulus, item stem, or answer option to display the context menu.
  2. Select the desired menu option.

Single-Button Mouse (for use with Mac computers)

Move the mouse to a desired element (a stimulus, item stem, or answer option).

  1. On the keyboard, press the [Ctrl] key and click the mouse button.
  2. Tap the [Menu] button in the upper-right corner. The menu will display the available options.
  3. Select the desired menu option.

Tablets

If you are using a tablet, you can use the on-screen [Menu] button to view the context menu for passages/stimuli and answer options. The button appears in the upper left corner (next to the zoom buttons).

  1. Tap on an element (a stimulus, item stem, or answer option).
  2. Tap the [Menu] button in the upper-right corner. The menu will display the available options.
  3. Select the desired menu option.

Chromebooks

Chromebooks do not have track pads with mouse buttons. If you are using a Chromebook that does not have a mouse plugged in, follow these instructions:

  1. Move the mouse pointer to a desired element (a stimulus, item stem, or answer option).
  2. Press and hold the [Alt] key on the keyboard.
  3. Press down on the track pad until it “clicks.” The context menu will open.
  4. Select the desired menu option.
