Overview of Test Tools

All tests have navigation and test tools. Some test tools are available only for specific tests or questions.

Sample Test Layout: Mathematics

The following test tools are available:

  • Menu Tools: Menu tools are those that appear at the top of the screen on the right side.
  • Question Tools: These tools are specific to questions.

Menu Tools

Menu Tools Description
Help [?] The [?] button at the top right will open the Test Information and Help window.
System Settings [*] The cog wheel [*] button at the top right will open the system settings window. You can change the volume of the sound in the test.
Calculator Click the [Calculator] button to open the on-screen calculator.
The calculator is available for the following tests: Grades 6–8 Mathematics; Secondary Mathematics I, II, and III; grades 4–8 Science; Biology; Chemistry; Earth Science; and Physics.
Formula Click the [Formula] button to open the on-screen Physics formula sheet.
Note: This reference sheet is available for Physics.
Masking To activate the Masking tool, click the [Masking] button. When the Masking button is orange, you can draw gray boxes on the screen to hide content.
When you are done masking, click the [Masking] button again.
Notes Click the [Notes] button to open an on-screen notepad. You can enter notes for yourself and save them. (Notes appear only for the test opportunity you are taking.)
Periodic Table Click the [Periodic Table] button to open the on-screen reference sheet.
Note: The Periodic Table sheet is available for Chemistry.
Questions drop-down list This feature enables you to quickly return to a specific question. Questions that were marked for review will display “(marked).”
Zoom buttons Use the [Zoom In] and [Zoom Out] buttons to increase and decrease the size of text and images. Four levels of magnification are available.

Question Tools

These tools require you to click on the context menu button [] for a stimulus (passage) or item. When you click on the context menu, a list of available options will appear. You can also right-click anywhere in a stimulus or item area to view the context menu.

Stimulus/Question Tools Description
Highlighter Highlight a section of text in a passage or test question.
Select text on the screen, click the context menu, and select [Highlight]. The selected text will become yellow.

Notes:
  • Text in images cannot be highlighted.
  • Highlighting is only available during a single test session. When you return to the test, any highlighting you did in a previous session will no longer be visible.
Mark for Review Click the context menu and select [Mark for Review]. The question number will change and show a white flap in the upper right corner.
Note: See the Pause Rules section below to learn about when you can review questions.
Stimulus Expansion Tool Some test pages have a passage or activity on the left side of the screen. You can expand this section.
  • Click the double arrow button [] to expand the left section so that it takes up most of the screen.
  • Click the double arrow button again to collapse the passage or activity.
Strikethrough For multiple-choice questions, you can "eliminate” an option and focus on what you think may be the right options.
  • Option 1: Select an answer option and then click the context menu and select [Strikethrough]. A line will appear through the text or image.
  • Option 2: Right click on an answer option and select [Strikethrough]. A line will appear through the text or image.
Tutorial Each question has a tutorial video that you can watch to learn more about that type of question. Click the context menu and select [Tutorial]. The video will open on the screen.

Basic Test Rules

  • You must answer each question. You can answer a question and mark it for review, then go back to it later to see or change your answer.
  • You must answer all questions on a page before moving on to the next page. (Some pages have a lot of information. You may need to use the vertical scroll bar to view all the information on a page.)
  • When you have reviewed your answers and finished the test, you must click the [End Test] button and follow the instructions to submit the test.
  • Once you submit the test, your score will be displayed.

Pause (Log Out) Rules

  • You can pause your test at any time. Pausing the test will log you out.
  • When you resume the test, you will see the first page that has unanswered questions.
  • If your test is idle for 20 minutes, the system will automatically pause the test and log you out.
For Math, Science, and ELA: Reading and Listening:
  • If your test is paused for less than 20 minutes, you can review questions you already answered.
  • If your test is paused for more than 20 minutes, you cannot review questions you already answered. The only exception is if you pause on a page that has at least one unanswered question.
For ELA: Writing:

Writing is divided into two sections, each with one written response required. You can pause the Writing sections at any time.

  • For the first essay: You can review and edit your response until you move on to the next section. Once you move on, you cannot go back.
  • For the second essay: You can review and edit your response until you are done and submit the test.

Text-to-Speech

You can listen to test instructions, passages, and questions as appropriate. For the ELA Reading/Listening items, some passages will not be read aloud.

  • To listen to a passage, click the passage context menu and select the [Speak] option you want. You can select a small portion of text, such as a word, phrase, or paragraph to listen to. Select the text, click the passage context menu, and select the [Speak Selection] option.
  • To listen to a question or answer options, click the question context menu and select the [Speak] option you want.
    • To listen only to the question, select [Speak Question] .
    • To listen to the multiple-choice question and all answer options, select [Speak Question and Options] .
    • To listen only to the answer option, click on the answer option select [Speak Option]. You can also right-click on the answer option and select [Speak Option]. (For example, if you want to listen to answer option A, right-click on that answer option and select [Speak Option A].)
