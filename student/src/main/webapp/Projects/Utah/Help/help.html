Test Rules

Basic Test Rules

  • You must answer each question. You can answer a question and flag it (mark it for review), then return to marked questions later to change your answer.)
  • You must answer all questions on a page before moving on to the next page. Some pages contain multiple questions. You may need to use the vertical scroll bar to view all questions on a page.

Pause Rules

  • You can pause your test at any time.
    • If your test is paused for less than 20 minutes, you can review questions you already answered.
    • If your test is paused for more than 20 minutes, you cannot review questions you already answered. The only exceptions are if you pause on a page that has at least one unanswered question or you are taking the Writing test.
  • When you resume the test, you will see the first page that has unanswered questions.
  • If your test is idle for 30 minutes, the system will automatically pause the test and log you out.

For Writing:

  • You can pause the Writing test for as long as you want. You can review your responses until you submit the test.
  • If your Writing test is idle for 30 minutes, the system will automatically pause the test and log you out.

About Text-to-Speech

In the secure browser, you can listen to test instructions, passages, and questions as appropriate. Some students may have Descriptive Audio as an accommodation in addition to text-to-speech.
Note for Reading, Language & Listening tests: Some passages will not be read aloud.

  • To listen to a passage, right-click anywhere on the passage and select the [Speak] option you want.
    Note: You can select a small portion of text, such as a word, phrase, or paragraph, and right-click to listen to that selection only.
  • To listen to a question or answer options, right-click on the question area or on an answer option and select the [Speak] option you want.
    • Select [Speak Question] to listen only to the question.
    • Select [Speak Question and Options] to listen to the multiple-choice question and answer options.
    • Select [Speak Option] to listen only to the answer option. (If you want to listen to answer option A, right-click on that answer option and select [Speak Option A].

Using Right-Click Features (Tablet and Single-Button Mouse)

If you are taking a test on a tablet or on a Mac computer with a single-button mouse, you can still access the following features:

  • Text-to-Speech
  • Highlighter
  • Strikethrough

Tablets

If you are taking a test on a tablet, you will see a "Menu" button in the upper left corner (next to the Zoom buttons).

Text-to-Speech and Highlighter

  1. Tap and select text that you want to listen to highlight, such as a paragraph or question.
  2. Tap the [Menu] button.
  3. Select the option that you want.

Strikethrough (Cross-Out Answer Options)

  1. Tap on an answer option.
  2. Tap the [Menu] button.
  3. Select "Strikethrough."

Mac Computers with Single-Button Mouse

If you are using a Mac computer that has a single-button mouse, you can still open the right-click menus.

  • On your keyboard, press the [Ctrl] key and click the mouse button.


Overview of the Student Testing Site and Test Tools


The following test tools are available:

  • Global Tools: Global tools are those that appear at the top of the screen in the black banner.
  • Stimulus/Question Tools: These tools are specific to stimuli and questions, and are accessed by clicking on an icon or right-clicking in the stimulus, question, or answer option areas.

Global Tools Description
Help [?] button The [?] button at the top left will open this Test Information and Help page.
Zoom buttons Use the [Zoom In] and [Zoom Out] buttons to increase and decrease the font size. Four levels of magnification are available.
Notes button Click the [Notes] button to open an on-screen notepad. You can enter notes for yourself and save them. (Notes appear only for the test opportunity you are taking.)
Calculator button Click the [Calculator] button to open the on-screen calculator.
Note: The calculator is available for grades 5–8 Mathematics and Secondary Mathematics I, II, and III tests, as well as grades 4–8 Science and Biology, Chemistry, Earth Science, and Physics tests.
Formula button Click the [Formula] button to open the on-screen Physics formula sheet.
Note: This reference sheet is available for Physics.
Periodic Table button Click the [Periodic Table] button to open the on-screen reference sheet.
Note: The Periodic Table sheet is available for Chemistry.
Questions drop-down list This feature enables you to quickly return to a specific test page. Questions that were marked for review will display “(marked).”
Save button This button allows you to manually save technology-enhanced questions. This is an optional feature. If you do not click this button, your answers will still be saved.
Note: This button will not appear on pages that have only multiple-choice questions.
Pause button This button allows you to pause the test. When you return to the test, you will be directed to the first page that has unanswered questions.
Navigation buttons The [Back] and [Next] buttons in the top right corner permit you to move between pages on a test.
End Test button This button appears after all test questions have been answered. Click this button to begin submitting the test.

Stimulus/Question Tools Description
Highlighter Highlight a section of text in a passage or test question.
Select text on the screen, right-click with the mouse, and select [Highlight]. The selected text will become yellow. Note: Text in images cannot be highlighted.
Mark for Review Each question has a flag icon. Click the flag to identify that question as one you may want to return to later. (The flag will turn dark blue and display a check mark.)
Note: See the Pause Rules section below to learn about when you can review questions.
Stimulus Expansion Tool Some test pages have a passage or activity on the left side of the page. You can expand this section.
  • Click the blue arrow icon [+ →] to expand the left section so that it takes up most of the screen.
  • Click the orange arrow icon [- ←] to collapse the passage or activity.
Strikethrough For multiple-choice questions, you can "eliminate” an option and focus on what you think may be the right options. Right click on an answer option and select “Strikethrough.” A line will appear through the text or image.
Tutorial Each question has a tutorial video that you can watch to learn more about that type of question. Click the round [ i ] button to open the tutorial video.

About Print-on-Request

If you are taking a test and have a Braille, large-print, or regular-print accommodation, you can request a printout of reading passages and/or questions.

  • To print a reading passage, click the [Print Passage] button. Your print request will be sent to your Test Administrator for approval.
  • To print a question, click the printer icon for that question. Your print request will be sent to your Test Administrator for approval.
