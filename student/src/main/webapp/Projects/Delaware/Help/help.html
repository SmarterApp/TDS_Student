Overview of the Student Testing Site


The following test tools are available:

  • Global Tools: Global test tools are those that appear at the top of the screen on the right side.
  • Context Menu Tools: Menu tools are specific to passages and items, and are accessed by clicking the context menu button [].

Global Tools Description
Help [?] The [?] button at the top right will open this Test Information and Help page.
Calculator Click the [Calculator] button to open the on-screen calculator. An on-screen calculator is available for the following tests:
  • Smarter Mathematics 6, 7, 8, and 11
  • EOC Algebra II
  • EOC Integrated Math III
Dictionary Click the [Dictionary] button to open the on-screen dictionary. The dictionary is available for the following tests:
  • Smarter ELA Performance Task
  • DCAS Science
  • DCAS Social Studies
  • EOC Algebra II, Integrated Math III, and U.S. History
Formula Click the [Formula] button to open the on-screen reference sheet. A formula sheet is available for the following tests:
  • EOC Algebra II
  • EOC Integrated Math III
Masking Click the [Masking] button to activate the Masking tool. You can use the tool to temporarily cover a section of the screen.
Note: This tool requires you to have the Masking accommodation.
Notes Click the [Notes] button to open an on-screen notepad. You can enter notes for yourself and save them. (Notes appear only for the test opportunity you are taking.)
Periodic Table Click the [Periodic Table] button to view an on-screen periodic table of elements. The Periodic Table is available for the following test:
  • DCAS Science 10
Zoom buttons Use the [Zoom In] and [Zoom Out] buttons to increase and decrease the size of text and images in the test. Four levels of magnification are available.

Context Menu Tools Description
Expand/Collapse
Passage		 Some test pages have a passage or activity on the left side of the page. You can expand this section.
  • Click the double arrow button [] to expand the left section so that it takes up most of the screen.
  • Click the double arrow button again to collapse the passage or activity.
American Sign
Language		 Watch a video of test content translated into ASL. Click the context menu and select [American Sign Language]. The video will open on the screen.
Note: This tool requires you to have the ASL accommodation.
This tool is available for the following tests:
  • Smarter ELA
  • Smarter Mathematics
Highlighter Highlight a section of text in a passage or test question.
Select text on the screen, click the context menu, and select [Highlight]. The selected text will become yellow.

Notes:
  • Text in images cannot be highlighted.
  • Highlighting is only available during a single test session. When you return to the test, any highlighting you did in a previous session will no longer be visible.
Mark for Review Click the context menu and select [Mark for Review]. The question number will change and show a white flap in the upper right corner.
Note: See the Pause Rules section below to learn about when you can review questions.
Strikethrough For multiple-choice questions, you can "eliminate” an option and focus on what you think may be the right options.
  • Option 1: Select an answer option and then click the context menu and select [Strikethrough]. A line will appear through the text or image.
  • Option 2: Right click on an answer option and select [Strikethrough]. A line will appear through the text or image.
Tutorial Each question has a tutorial video that you can watch to learn more about that type of question. Click the context menu and select [Tutorial]. The video will open on the screen.

Test Rules

Basic Test Rules

  • You cannot skip questions. (You can mark a question for review and return to it later.)
  • You must answer all questions on a page before moving on to the next page. Some pages contain multiple questions. You may need to use the vertical scroll bar to view all questions on a page.
  • You may review past questions and change your answer (see Pause Rules below).

Pause Rules

  • You can pause your test at any time. You will be presented with a warning message asking you to verify that you want to pause the test.
    • If your test is paused for less than 20 minutes, you can review items you already answered.
    • If your test is paused for more than 20 minutes, you cannot review items you already answered. (The only exception is if you pause on a page that has at least one unanswered item.)
  • When you resume the test, you will see the first page that has unanswered items.
  • If your test is idle for 30 minutes, the system will automatically pause the test and log you out.
  • All answers are saved immediately; you do not lose them when test is paused.

Navigation Rules

  • You cannot move forward in the test until all items on the current page have been answered.
  • You can navigate to any available test page by clicking the [Back] or [Next] buttons or selecting a page from the "Questions" drop-down list at the top of the page.

About Accommodations

About Print-on-Request

If you are taking a test and have a print accommodation, you can request a printout of test content.

  • To print a passage, click the [Print Passage] button in the Global Menu. Your print request will be sent to your Test Administrator for approval.
  • To print a question, click the question context menu and select [Print Item]. Your print request will be sent to your Test Administrator for approval.
  • To print a passage and all questions on the page, click the [Print Page] button in the Global Menu. Your print request will be sent to your Test Administrator for approval.

Text-to-Speech

You can listen to test instructions, passages, and questions as appropriate. For ELA, some passages will not be read aloud.

  • To listen to a passage, click the passage context menu and select the [Speak] option you want. You can select a small portion of text, such as a word, phrase, or paragraph to listen to. Select the text, click the passage context menu, and select the [Speak Selection] option.
  • To listen to a question or answer options, click the question context menu and select the [Speak] option you want.
    • To listen only to the question, select [Speak Question].
    • To listen to the multiple-choice question and all answer options, select [Speak Question and Options] .
    • To listen only to the answer option, click on the answer option select [Speak Option]. You can also right-click on the answer option and select [Speak Option]. (For example, if you want to listen to answer option A, right-click on that answer option and select [Speak Option A].)
